== Description ==
Fixes up products for SG Motorsport


== Installation ==
1. Upload the contents of the sgfixup plugin to the `/wp-content/plugins/sgfixup' directory
2. Upload oik-batch
3. Activate the sgfixup plugin through the 'Plugins' menu in WordPress
